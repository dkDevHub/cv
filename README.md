## 📖 Про проєкт

**CV** — це не звичайне резюме у форматі PDF, а живий інтерактивний застосунок, що демонструє реальні навички frontend-розробки. Замість переліку скілів у тексті — робочі приклади коду, анімації та готові UI-компоненти, зібрані в єдиний UI Kit.

Проєкт покликаний показати не просто "що я знаю", а **як саме** це реалізовано на практиці.

## ✨ Можливості

- 🏠 **Hero-секція** з коротким представленням та відгуками
- 🧩 **Tech Stack** — детальний розбір основних технологій (TypeScript, React, Next.js, Zustand, Tailwind) з прикладами коду
- 👤 **About me** — освіта, досвід та поточний фокус розвитку (Backend на Bun.js, Astro JS, англійська B2)
- 🎨 **UI Kit** — набір власноруч написаних UI-компонентів:
  - Range Select (слайдер діапазону цін)
  - Кастомні кнопки (Neutral / Outline / Ghost / Primary / Icon)
  - Модальні вікна
  - Поле вводу номера телефону з маскою
  - Сортування та фільтри
  - Dropdown-списки
  - Лічильник кількості
- 🔌 **Інтеграції з реальними API**:
  - Nova Poshta — вибір міста та відділення
  - Binance Spot — курси криптовалют у реальному часі
  - GitHub — пошук користувачів
- 📱 Повністю адаптивний дизайн

## 🚀 Швидкий старт

### Вимоги

- Node.js 18+
- npm

### Встановлення

```bash
git clone https://github.com/dkDevHub/cv.git
cd cv
npm i
```

### Запуск у режимі розробки

```bash
npm run dev
```

Застосунок буде доступний за адресою [http://localhost:3000](http://localhost:3000).

### Збірка для продакшену

```bash
npm run build
npm run start
```
